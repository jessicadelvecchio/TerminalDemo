Arrows scroll the page

7. What is the difference between a terminal and shell?
Shell is the program, a terminal is where the software is run, similar to a browser


8. What is an absolute path?
An absolute path is the direct path to a directory. This can be found by the pwd command. The path will start at /Users/.... And continue to the desired directory.


9. What is a relative path?
A relative path is the path to the desired directory based on your current location. 


10. What is a flag? Give three examples of flags you have used.
A flag is a condition that can be added on the existing commands. 
Ls -a will show all items in a folder including those that start with . 
Ls -l will show additional details for all contents of a directory
Rm -rf will delete a directory will all of its contents


11. What do the r and f flags do with the rm command?
-f will remove files without prompting for confirmation regardless of the file's permissions
-r will remove the file hierarchy rooted in each file argument	

-rf will remove a directory with contents
